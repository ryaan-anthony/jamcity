# jamcity
Run TidalCycles and SuperCollider from the command line.

### Prerequisites

- [TidalCycles/SuperCollider](https://tidalcycles.org/index.php/Installation)
- [Haskell platform](https://www.haskell.org/platform/)
- Ruby >= 2.4


### Start SuperCollider:
```
bin/super_collider
```

### Run TidalCycles:
```
bin/tidal_cycles
```

### Test sound:
```
d1 $ sound "bd*2 cp"
```
